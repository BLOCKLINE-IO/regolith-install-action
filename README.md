# regolith-install-action
GitHub Action to setup Regolith in your workflow

Usage in workflow yml:
``` 
# some code...
steps:
  - uses: BLOCKLINE-IO/regolith-install-action@v1.0
    with:
      version: 0.0.17 # see action.yml for default version
```
